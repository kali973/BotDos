Flow Browser

Instructions

To create a real-time traffic flow trend chart:

  1. Keys Enter a comma separated list of packet attributes to define the flow, e.g. ipsource,ipdestination
  2. Value Enter a value to plot, e.g. bps (Bits per Second)
  3. Filter Enter a filter to select specific traffic, e.g. ipprotocol=6 (TCP)
  4. Submit Click the button to create the chart

See Defining Flows for more information

Click on peaks in the trend chart to see values at that time. Click on items in the chart legend to drill down by appending the item to the current filter.

Click on Clone button to create a new window with current settings. Bookmark/copy URL to re-create chart.
